# DSA-210 Project

## Overview

This project analyzes wind electricity market data (2014–2021) using Exploratory Data Analysis (EDA) and hypothesis testing.

## Contents

* Python code: data cleaning, visualization, hypothesis testing, and regression
* Word report: visualizations and results


* Relationship between MCP and SMP prices
* Differences in prices across time groups (Night, Morning, Afternoon, Evening)
* Monthly seasonality analysis
* Statistical hypothesis testing (ANOVA, correlation, regression)

## How to Run

1. Open the Python script or notebook
2. Install required libraries (pandas, matplotlib, scipy, statsmodels)
3. Run the code to generate graphs and results

## Notes

* Dataset: Wind electricity market data (2014–2021)
* All outputs are saved in the `project_outputs` folder
